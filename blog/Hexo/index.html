Hexo

一个基于Node.js 的静态网页生成器，常将它与Github/Gitee搭配使用，创建个人博客网站。
特点：可以自由更换不同风格的Blog，在官网插件可进行下载

本Blog是基于Butterfly 风格搭建的，详细配置可参考Butterfly官方文档

常用指令：
hexo s： 启用本地服务器，浏览调试主题
hexo n “标题”： 新建文章
hexo d： 自动生成网站静态文件，并提交至Github/Gitee
hexo clean: 清除缓存：db.json 以及已生成的静态文件：public
hexo g： 生成网站静态文件到public
hexo new page xxx：新建一个标题为xxx 的页面，默认链接地址为 主页地址/xxx/
Attention ：hexo n 时，标题可以为中文，但一般习惯用英文；页面标题和文章一样可以随意修改；但页面不会出现在首页文章列表和归档中，也不支持设置分类和标签

点击查看如何快速本地布建一个Blog

详细请看Hexo官方指令

Bat批处理文件

本质：DOS(Disk Operating System,即磁盘操作系统)命令集合。
扩展名一般为：.bat/.cmd
可用if/goto控制流程，也可用for循环，大小写不敏感

基本命令：

  • echo on/off 内容:显示命令，后可跟字符串、变量等，与Linux中echo相同
  • @：用于隐藏其后的命令 如关闭回显：@echo off(包括本身)
    goto 参数：跳转到特定行并执行下一行开始的命令，参数为字母时，前面要加“：”表示是参数，与if配合使用
  • Rem 内容（常用::代替rem）：注释
  • Pause：暂停批处理并显示cmd界面
  • call=批处理路径 参数：调用另一个批处理程序而不关闭当前程序，传递参数（可调用自身：死循环、递归）
  • start 文件路径 参数：调用外部程序，并传递参数，所有的 DOS命令 和 命令行程序 都可以由 start命令 来调用
  • if “参数”==字符串:判断条件并执行不同命令

常用Windows命令：

  • cd:切换目录
  • dir：显示当前目录内容
  • tree:显示目录结构
  • ren 文件路径 更改名:重命名目录/文件
  • md:创建目录
  • cls:清屏

网络测试常用：
- ipconfig:查询IP 如需更详细信息则后+/all
- ping:测试网络联通性,但不能测端口，利用ICMP
- nslookup:域名查IP

  • tracert:查询网络路由(本机IP到目的IP经过的路由情况)
  • arp -a:查询高速缓存中所有项目
  • route print：显示路由表中当前项目
  • nbtstat -n:查询本机网络配置
  • netstat:显示网络当前的状态(TCP、监听端口等)
  • 命令 man:显示命令的语法
  • net help:多种用法

Win+R常用：

  • control：控制面板
  • mstsc：远程控制电脑
  • msconfig：系统配置
  • services.msc:服务
  • devmgmt.msc:设备管理
  • regedit：注册表
  • appwiz.cpl:卸载程序
  • wf.msc:防火墙
    Win+E：文件管理
    Win+D：显示桌面

Markdown基本语法

Markdown是一种纯文本格式的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。

写法:

# 这是一级标题

## 这是二级标题

### 这是三级标题

#### 这是四级标题

##### 这是五级标题

###### 这是六级标题
