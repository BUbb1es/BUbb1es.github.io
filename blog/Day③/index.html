后端

类似仓库(存储数据)
数据库服务软件与SQL关系：
SQL：结构化查询语言，也是ANSI计算机语言(可访问&操作数据库),大小写不敏感 ,’;’表示语句结束
数据库：兼容SQL关键词操作(MySQL Oracle SQLserver)

  • 数据库
    环境：MySQL(软件)、端口/本地(服务接口)、root等(用户)

常用操作：
创建数据库 CREATE DATABASE XXX;
查看数据库 SHOW databases;
切换数据库 USE XXX;
删除数据库 DROP DATABASE XXX;

Key(键) 1表头属性 2表头属性
1属性集合 2属性集合 3属性集合

行：一条数据，内容可划分为数据块(值)
列：表头对应的属性集合
键：用于识别每行数据(唯一)

  • 数据表
    常用操作：
    创建数据表

    查看所有数据表 SHOW tables;
    查看特定数据表 SELECT * FROM XXX;
    删除数据表 DROP TABLE XXX;
    导入数据表 SOURCE SQL文件路径;(\要2个,因为运行过程会去掉其一\)

  • 数据(增删改查&软件内置函数)
    常用操作：
    插入数据表数据 INSERT INTO XXX(变量2，变量3) VALUES(‘新数据’,’新数据’,’新数据’);
    删除数据表数据 DETELE FROM XXX WHERE 变量2 = ‘想删除的数据’;
    更新数据表数据 UPDATE XXX SET 变量2 = ‘新数据’ WHERE 键 = XX;
    查看特定数据表数据 SELECT * FROM XXX WHERE 变量1 = ‘查询数据1’ and 变量2 = ‘查询数据2’;

常用筛选语句：
SELECT 所需信息 FROM 数据表(1或多) WHERE 筛选条件;

常用排序语句:
SELECT 所需信息 FROM 数据表(1或多) ORDER BY 变量 ASC(升序)/DESC(降序);
ATTENTION : 可用 order by 数,判断数据表变量长度

合并不同表数据的语句：
SELECT 变量1 FROM 数据表1 UNION SELECT 变量1 FROM 数据表2;(不显示 不同表内重复的数据)
SELECT 变量1 FROM 数据表1 UNION ALL SELECT 变量1 FROM 数据表2;(可显示不同表内重复的数据)

语句解释：

  • MySQL内置函数：
    使用SELECT运行
    database(); 打印当前所用数据库名称
    current_user; 打印当前用户
    load_file(特定文件路径); 返回特定路径文件的内容
    version(); 打印当前数据库版本
    into+outfile可写入文件

  • PHP(Hypertext Preprocessor)
    超文本预处理器，开源脚本语言，用于网页开发，服务器上运行

  • 基本架框：基础语法、语句、变量、函数

    Attention:

  • 文件包含
    想将两个php文件显示同一页面内：
    include ‘文件名’; 有警告也会继续运行
    require ‘文件名’; 若错误，停下脚本


  • 使用数据库(MySQL)
  1. 连接服务器，即MySQL服务： $连接变量 = mysql_connect(“服务器名称”,”用户名”,”密码”)
  2. 设定编码(防乱码)：mysql_query(“SET NAMES UTF8”);
  3. 选数据库：mysql_select_db(“数据库名称”,$连接变量);
  4. 操作数据库： mysql_query(“SELECT * FROM 当前数据库名称”);
  5. 关闭数据库： mysql_close($连接变量);
